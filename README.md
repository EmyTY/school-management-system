# Информационна система за дигитализиране на учебно заведение

**Курсова работа** по дисциплината **"Проектиране на информационни системи"**  
Икономически университет – Варна

## Описание на проекта

Система за управление на училищни дейности, която включва:
- Управление на ученици и учители
- Въвеждане и проследяване на оценки
- Отчитане на присъствия
- Аналитични справки и визуализации (Power BI)

## Технологии

- **База данни**: SQL Server / SQLite
- **Моделиране**: Erwin Data Modeler
- **Визуализации**: Power BI Desktop
- **Диаграми**: UML (Use Case, Class, Sequence, Activity)

## Съдържание на хранилището

- `docs/` — Пълният курсови проект (PDF) + ER диаграма
- `database/` — SQL скрипт за създаване на базата + примерни данни
- `PowerBI/` — Аналитичен отчет

## Основни функционалности

- Ролево-базиран достъп (Администратор, Учител, Ученик)
- Въвеждане на оценки и присъствия
- Генериране на справки за успеваемост
